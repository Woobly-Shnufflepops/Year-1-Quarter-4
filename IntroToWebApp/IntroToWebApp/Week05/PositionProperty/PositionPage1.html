Positioning elements

To position elements on a page, use the position property. This property can be set to:
relative, absolute, fixed, or static

Once the position property is set, you could use the top, bottom, left, right properties.

relative positioning
The element is positioned relative to its normal position. So left:50px adds 50 pixels to the element's left position relative to its normal position
