Javascript Timers

Two timers are defined:
*one-shot timer: setTimeout(function, miliseconds) *Interval timer: setInterval(function, miliseconds)

the setTimeout starts the function after a certain time, stated in miliseconds, has elapsed.
The setInterval runs the function every intervaltime, as stated in miliseconds.

Both timers return an id that can be used to clear the timer
using the clearTimeout(id) and clearInterval(id)
