The float Property

This property is used for positioning and layout on web pages.

The property can be assigned 4 values:
left: the element floats to the left of its container
right: the element floats to the right of its container
none: the element does not float. This is the default value
inherit: the value is inherited from its parent

You can use it to layout div tags next to each other horizontally.
You can also use it to have text wrap around an image

The overflow property

This property specifies whether to clip content or to add scrollbars when the content of an element is too big to fit in a specified area of an element, such as a div o p tag

This property can be set to the following values:
visible: Default: overflow is not clipped. It renders outside the box hidden: The overflow is clipped, and the rest of the content is visible scroll: the overflow is clipped, but the scrollbar is added to see the rest of the content
auto: same as scroll, but scrollbar is added only when needed.
