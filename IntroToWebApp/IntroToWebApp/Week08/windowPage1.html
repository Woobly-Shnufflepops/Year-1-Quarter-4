Javascript Browswer BOM

BOM stands for Browser Object Model.
The BOM defines a set of precreated objects such as: window, screen, location...

See link: js browswer BOM

Javascript HTML DOM

DOM stands for: Document Object Model.
The browswer takes an HTML document and builds a tree hierarchy of all the elements in the HTML. Ever element is considered an object and therefore has properties and methods.

There are other objects in the browswer such as:
Date, array, math, string

The document object

The document.getElementById("id") returns a reference to the first element with the given id, then use its properties to access or alter its content
