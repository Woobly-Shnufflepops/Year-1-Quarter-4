The Class Selectors

You can define style rules based on the class attribute of the elements. All the elements having that class will be formatted according to the defined rule.

The ID Selector

You can define style rules based on the id attribute of the elements.Since it is preferable to define unique id for each element you can use the id as a selector to style a specific element, such as styling some p tag among many p tags
